# 場次／隊伍匯入 — Excel（xlsx）格式說明

## 使用時機

1. 在後台建立 **賽事（Tournament）**  
2. 在同一賽事內先建立 **組別**；**隊伍**可手動新增或見下方 **隊伍匯入**  
3. 場次 Excel：隊伍名稱須與後台**完全一致**（含匯入隊伍）  
4. 於 **賽事管理頁** 上傳檔案（讀取**第一個工作表**）

> **淘汰賽預排（小組後淘汰）**：若你想先排好淘汰賽時間與對賽，可在淘汰賽賽事匯入場次時，於 `team_a` / `team_b` 欄填入 **slot**（例如 `A1`、`B2`）或 `BYE`，系統會建立占位隊伍並保留場次時間/場地。待小組賽結束後，再用後台「由小組賽產生淘汰賽」一鍵把 `A1/B2…` 填入實際出線隊伍。

---

## 隊伍匯入

在賽事管理頁「**Excel 匯入隊伍**」上傳；建議先建立 **組別**，再在表內填組別名稱。

### 第一列：表頭（擇一寫法）

| 建議英文表頭 | 必填 | 說明 |
|--------------|------|------|
| `team_name` 或 `name` | **是** | 隊伍名稱（同一賽事內不可重複；已存在則該列略過） |
| `group` | 否 | 組別名稱（須與後台組別**完全一致**；可留空＝未分組） |
| `seed` | 否 | 種子序號，非負整數（可留空） |

也支援中文表頭別名：**隊伍**、**隊伍名稱**、**隊名**、**組別**、**種子**。

### 注意

- 空白列會略過  
- 檔案內或資料庫已存在的同名隊伍會略過並在結果中提示  
- 填了組別名但後台找不到該組別時，該列**不會建立**  
- 匯入為**新增**隊伍，不會刪除既有隊伍

### 範例

```
team_name | group | seed
紅隊      | A組   | 1
藍隊      | A組   | 2
```

---

**日期**由 **大會設定** 的起訖日涵蓋；場次僅需填 **開賽時間（時:分）**。

## 第一列：表頭（擇一寫法）

| 建議英文表頭 | 必填 | 說明 |
|--------------|------|------|
| `team_a` | **是** | 隊伍 A 名稱（與後台隊伍名一致） |
| `team_b` | **是** | 隊伍 B 名稱 |
| `match_format` | 否 | 預設 `bestOf3`。見下方「賽制」 |
| `round` | 否 | 輪次／標籤，例如：`A組`、`八強` |
| `court` | 否 | 場地，例如：`1號場` |
| `scheduled_time` | 否 | **僅時間** `HH:mm`（例如 `14:00`、`09:30`）。也可用 `scheduled_at` 表頭 |
| `group` | 否 | 組別名稱（須與後台組別**完全一致**） |

也支援中文表頭別名，例如：**隊伍A**／**隊伍B**、**賽制**、**輪次**、**場地**、**開賽時間**、**組別**。

## 賽制（`match_format`）

| 值（英文） | 中文 |
|------------|------|
| `bestOf3` | 三局兩勝 |
| `bestOf5` | 五局三勝 |
| `singleGame` | 一局過 |

## 開賽時間（`scheduled_time`）

- 填 **24 小時制時分**：`14:00`、`09:30`  
- Excel 儲存格設為「時間」格式亦可（系統會換算）  
- 可留空  
- **不要**再填完整日期時間（大會日期已在大會設定）

## 範例（第二列起為資料）

```
team_a | team_b | match_format | round | court | scheduled_time | group
紅隊   | 藍隊   | bestOf3      | A組   | 1號場 | 14:00          | A組
```

## 注意

- 空白列會略過  
- 找不到隊伍名、組別名時，該列**不會建立**並於匯入結果中列出原因  
- 匯入為**新增**場次，不會刪除既有手動建立的場次
